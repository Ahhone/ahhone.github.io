
# Image Assets for Mijatov D.O.O Website

## Overview

This folder contains all image assets used throughout the Mijatov D.O.O website. These images are referenced in various components including the hero section, product carousels, brand partnerships, and company branding elements.

All images should be stored locally in this directory to ensure optimal performance, faster load times, and reduced dependency on external CDN services.

## How to Download and Place Images
